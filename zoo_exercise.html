Illustrator practice: zoo map

Feb. 2, 2020

This semester, I am working on the graphics desk at The Columbia Missourian, where we use Adobe Illustrator for almost all of our visualizations. While I don't love it as a data visualization tool, I do like it for creating vector art. It can be kind of finicky, so I wanted to practice using it for map making.

Goal: Create a basic map of the San Diego Zoo

Successful?: Yes


Tools used:

Adobe Illustrator

Task sequence

  1. Find map of San Diego Zoo online
  2. Open image in Adobe Illustrator
  3. Create a new layer and trace outlines of different park sections
  4. Label map sections
  5. Trace logo using same method as above
  6. Smooth polygons
  7. Voila!

Final results

Conclusion

Adobe Illustrator is effective in replicating map sections as polygons.

Read more

While I find Adobe Illustrator almost counterintuitive to use if you are working with data in a spreadsheet format, it is nice for making vector art.


Something I did spend a bit of time on was using the "smooth" tool, which is accessed after selecting a polygon, clicking on the paintbrush tool (B), and holding down ALT/OPTION while tracing jagged lines. I think this made some elements more visually pleasing, though I do fear that when used too much it might alter the accuracy of the trace because the lines will no longer line up exactly.


There's nothing especially earth-shattering about this exercise, I just wanted to practice using Illustrator some more. I chose the San Diego Zoo because I went there ALL! THE! TIME! growing up. I always liked the panda exhibit and remember the polar bears always being asleep. Check it out if you're every in the area!

The Creation of Playground

Jan. 12, 2020

I'm so excited to be typing these words right now, because it means I have successfully (finally) created Playground! If you've visited my 'About' page, you may already know what Playground is. In short, it is a place for me to document the things I do in my free time having to do with data, computers, data viz, programming and anything else that excites me. In this first post, I am going to walk you through how I created Playground, from start to finish.


Let's start by looking at the tools I used:

HTML

CSS

Github/Git

HTML

The foundation of this site, as with most sites, is HTML, or the structure of elements on a webpage. Playground was coded by me from scratch in Sublime Text, which is a free text editor where you can write computer code and save it as a file on your computer. One of the reasons I like Sublime Text is because you can easily preview your code by right-clicking and selecting "open in browser."


All of the pages on Playground are structured with Materialize grids.


The navigation bar at the top of earch page was created using this tutorial from W3Schools, which is one of my favorite resources for all things HTML and CSS. The navigation bar is a mixture of HTML, CSS, and JavaScript elements.


The trickiest piece of HTML on this site thus far has been the tool circles seen on this page. The position of the circles directly above the text was not something I was able to do using a "text-align" attribute. This is frustrating because it means they may look different depending on what type of browser or device someone is viewing the page on.
